# gefanimig.se

Free tool for removing your personal data from Swedish databases using GDPR.

**https://gefanimig.se**

## What it does

- Direct links and instructions for 7 Swedish databases (Ratsit, Merinfo, Hitta, Eniro, etc.)
- Ready-made GDPR email template
- Google search result removal guide
- IMY complaint info if a company refuses

## Stack

Nuxt 4, Tailwind CSS, TypeScript. Hosted on Fly.io. Analytics via self-hosted Umami.

## Dev

```bash
cp env.example .env
npm install
npm run dev
```

## Deploy

```bash
flyctl deploy
```
